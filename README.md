# StyleMate AI - Your Personal AI Fashion Stylist 👗✨

StyleMate AI is a premium fashion-tech assistant designed to integrate seamlessly into the fashion ecosystem. Inspired by industry leaders like Myntra, Zara, and Apple, it offers a sophisticated, minimalist interface for personalized outfit recommendations.

## ✨ Features

- **AI-Powered Styling**: Get curated outfit suggestions for various occasions (Weddings, Farewells, Interviews, etc.).
- **Premium Design**: A clean, modern aesthetic with a soft blush pink palette (#FF3F6C).
- **Interactive Experience**: Smooth animations using Framer Motion and micro-interactions.
- **Product Curation**: View detailed recommendation cards with images, pricing, and ratings.
- **Responsive Layout**: Optimized for desktop and mobile fashionistas.

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Typography**: Inter

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run the Development Server**:
   ```bash
   npm run dev
   ```

3. **Open the App**:
   Navigate to the local URL provided in the terminal (usually `http://localhost:5173`).

## 💡 How to Use

- Select a **Quick Action Card** (e.g., "Farewell Outfit") to see a curated recommendation.
- Type your requirements into the chat (e.g., "What should I wear for an interview?").
- Hover over product cards to see the "Quick View" animation.

## 📝 Demo Notes

This is a high-fidelity frontend prototype built for the Myntra Hackathon demo. The AI responses are currently backed by realistic mock data in `src/data/mockResponses.js`.
